亻 
ideographs

  connotation : general subset
  originates from : 亻
  originally means : person, people, human
As component of ideographs, it implies The type of output data is same as the input. And the elements of output data are part of the input. In particular, 化 is exceptionally different, which has a more diverse usages scenarios, 化Ix, 化Sx. Note that, if the procedure name has another ideograph prefix it as a word, the input and output type is determinated by the prefix ideographs, e.g.,  􏻿𢪛 􏿰攸 􏻿弔 􏿰弔 􏻿巨 􏿰巨.

Examples:  伄 攸 𰂋 偏 􏾜 􏾛 偅 𠆯 􏹈.

  connotation : strengthen
Almost has the same usage as the one named without *, except this strengthen the processing of inputted data. Thus the data of output may become longer, and the type may be changed accordingly.
